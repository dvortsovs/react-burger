# Проект: Stellar Burgers
---
Веб приложение для оформления заказов еды, созданное в качестве учебного проекта для Яндекс.Практикум.

---

## В проекте использованы следующие технологии:

+ React
+ Redux
+ React-router
+ React-DND
+ WebSocket
+ CRA
+ Ya.Praktikum UI
